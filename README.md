# Describe project


## SQL - Prisma
### Update schema database
For development mode
(recommended):
```
 npx prisma db push  
```
or
```
npx prisma migrate dev --name init
```

For product mode:
```
 npx prisma migrate  
```

### Seeding database
https://www.prisma.io/docs/orm/prisma-migrate/workflows/seeding
```
npx prisma db seed
```

### Explore the data in Prisma Studio (browser)
```
npx prisma studio 
```
